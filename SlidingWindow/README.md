# Sliding Window Pattern

## What is Sliding Window?
Sliding Window is a technique used to reduce time complexity by avoiding repeated work on overlapping subarrays.

Instead of recalculating values for every window, we reuse previous computations.

## When to Use
- Subarray or substring problems
- Fixed or variable window size
- Maximum / minimum / sum type problems

## Types
1. Fixed Window Size
2. Variable Window Size

## Problems Covered
- Maximum sum of subarray of size k
- First negative number in every window
- Longest substring without repeating characters

## Purpose
To improve efficiency from O(n²) to O(n)
