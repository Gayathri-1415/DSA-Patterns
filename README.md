# DSA Patterns (Java)

## Overview
This repository contains Data Structures and Algorithms (DSA) problems solved using a pattern-based approach.

The goal is to understand problem-solving techniques by grouping similar problems under common patterns.

## Patterns Covered
- Sliding Window
- Two Pointers
- Recursion
- Backtracking
- Stack
- Queue
- Binary Search

## Structure
Each folder represents a specific pattern and contains:
- A README explaining the concept
- Java programs for related problems


## Purpose
- To build strong problem-solving skills
- To understand common DSA patterns
- To prepare for coding interviews

## Language Used
Java
